The goal of this project is to enable users to quickly provision a Kali machine into an attack workstation suitable for attack simulation, install the necessary tools along with our Python interface, and thereby rapidly and semi-automatically execute attack chains generated by Aurora.

## One-Click Automated Deployment
Users can choose to run this script to automatically provision a Kali machine into an attack workstation capable of executing scripts generated by Aurora.

```bash
bash auto_deploy.sh
```

## Manual Step-by-Step Deployment

```bash
mkdir attack_tools && cd attack_tools
wget https://github.com/BishopFox/sliver/releases/download/v1.7.2/sliver-server_linux-amd64
```
